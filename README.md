# 海南医科大学沉浸式招生网站

## 预览

- 本地静态服务：`http://127.0.0.1:57252/`
- 直接打开：`index.html`

## 数据说明

- `data/site.json`：学校简介、官网链接、二维码占位信息。
- `data/gallery.json`：校园美图、冷暖色调标记、来源链接。
- `data/majors.json`：27 个当前本科招生展示专业卡片及综合实力排序依据；专科专业和当前不再作为本科招生展示的专业不在页面展示。
- `data/admissions.json`：已整理的公开录取数据，当前覆盖 31 个省份、2016-2025 年、3949 条专业分数/位次记录。缺失专业/年份会在页面显示暂无数据，不做虚构。
- 校名沿革展示规则：2024 年以前的录取数据对应“海南医学院”，2024 年及以后的录取数据对应“海南医科大学”；来源标题保留原网页表述。
- 已按“海南医学院 / 海南医科大学”两个校名检索并合并数据；主要来源：海南医科大学招生系统、海南医学院 2016-2019 年官方分省专业工作簿、夸克高考公开 OAPI（中国教育在线/掌上高考专业分数线）、大学生必备网、高考网、高考100、大考100、高考升学网、大学加油网、大学志、6617、零二七艺考、果然优志分省分年页面。
- 历史整理但不再展示的专业及录取记录已归档至 `work/removed_noncurrent_majors.json`。

## 后续替换

- 新生群二维码位置在首页欢迎区和专业详情侧栏，当前为空白占位。
- 继续补充录取数据时，保持 `data/admissions.json` 字段一致，并同步更新 `app.js` 内置 DATA 以支持直接打开文件。
